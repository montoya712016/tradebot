# Astra Tradebot Logo Concepts

Propostas vetoriais iniciais para a marca `Astra Tradebot`.

Arquivos:
- `astra-tradebot-logo-orbital-a.svg`
- `astra-tradebot-logo-orbital-star-depth.svg`
- `astra-tradebot-logo-orbital-star-elegant.svg`
- `astra-tradebot-logo-orbital-star-signature.svg`
- `astra-tradebot-logo-orbital-star-signature-aurora.svg`
- `astra-tradebot-logo-orbital-star-signature-noctis.svg`
- `astra-tradebot-logo-orbital-star-signature-mono.svg`
- `astra-tradebot-logo-orbital-star-signature-ember.svg`
- `astra-tradebot-logo-orbital-star-signature-crimson.svg`
- `astra-tradebot-logo-orbital-star-signature-amber.svg`
- `astra-tradebot-logo-orbital-star-trails-7.svg`
- `astra-tradebot-logo-premium.svg`
- `astra-tradebot-logo-north-star.svg`
- `astra-tradebot-logo-signal-mark.svg`

Ideias:
- `orbital-a`: monograma `A` com orbita parcial
- `orbital-star-depth`: estrela central com orbita inclinada, separando o segmento traseiro e o frontal para dar profundidade
- `orbital-star-elegant`: mesma ideia orbital, mas com estrela central mais contida e institucional
- `orbital-star-signature`: mesma estrutura orbital, mas com paleta mais propria da Astra, reduzindo o azul generico
- `signature-aurora`: assinatura com mais iris e contraste, ainda equilibrada
- `signature-noctis`: assinatura mais escura e premium
- `signature-mono`: assinatura puxando mais para mint/slate, menos violeta
- `signature-ember`: familia cobre/ember, mais quente e executiva
- `signature-crimson`: familia vinho/vermelho queimado, mais ousada sem virar neon
- `signature-amber`: familia dourado/ambar, mais solar e institucional
- `trails-7`: uma orbita principal com seis trilhas finas adicionais para simular rastro orbital
- `premium`: versao mais limpa e corporativa, com uma unica orbita dominante e estrela simplificada
- `north-star`: estrela tecnica de navegacao
- `signal-mark`: simbolo mais modular e sistematico

Todos os arquivos sao SVG editaveis e podem ser refinados depois.

Uso atual:
- a landing publica compartilhada passou a usar `astra-tradebot-symbol-premium.svg` como mark default do mock institucional
- o `astra-tradebot-logo-premium.svg` continua disponivel como wordmark completo para usos futuros
